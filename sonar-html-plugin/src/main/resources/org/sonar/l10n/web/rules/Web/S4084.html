In order to make your site usable by as many people as possible, subtitles should be provided for videos.

This rule raises an issue when a video does not include at least one subtitle <track/>.

Noncompliant Code Example

<video id="video" controls preload="metadata">
   <source src="resources/myvideo.mp4" type="video/mp4">
   <source src="resources/myvideo.webm" type="video/webm">
</video>

Compliant Solution

<video id="video" controls preload="metadata">
   <source src="resources/myvideo.mp4" type="video/mp4">
   <source src="resources/myvideo.webm" type="video/webm">
   <track label="English" kind="subtitles" srclang="en" src="resources/myvideo-en.vtt" default>
   <track label="Deutsch" kind="subtitles" srclang="de" src="resources/myvideo-de.vtt">
   <track label="Espa√±ol" kind="subtitles" srclang="es" src="resources/myvideo-es.vtt">
</video>
